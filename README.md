# AutoElite Motors

AutoElite Motors is a web application for showcasing and managing vehicle listings for a car dealership. It allows users to browse available vehicles, view detailed specs and pricing, and helps dealers manage their inventory with an easy-to-use interface.

## Features
- Browse new and pre-owned vehicle listings
- Search and filter by make, model, price, and year
- Detailed vehicle pages with specs and images
- Admin dashboard for inventory management
- Responsive design for mobile and desktop
